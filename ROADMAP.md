# Roadmap

## v0.1

- Context Markdown format
- CLI index, search, relevance, lint, export
- GitHub Action PR comment
- MCP server with search, relevance, and lint tools
- Example central context repo

## v0.2

- Context update proposal command for merged PRs
- Better diff parsing and symbol extraction
- Stale context detection using git history
- CI annotations for blocking context violations
- More adapter exports for Cursor and Copilot instructions

## v0.3

- Optional embeddings index
- Web UI for browsing context, owners, and lineage
- Context conflict detection
- Multi-repo manifest for large organizations
- Signed context snapshots for regulated teams

## Non-goals

- Replacing code review
- Auto-approving durable engineering guidance
- Becoming a hosted-only service
- Stuffing the entire context base into `AGENTS.md` or `CLAUDE.md`
